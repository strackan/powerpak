# MCP Universe Mockups - Quick Start

## Open the Demo in 3 Steps

1. Navigate to: `C:\Users\strac\dev\MCP-World\mockups\`
2. Double-click: `marketplace-ui/index.html`
3. Start exploring!

## What You'll See

### Main Marketplace (`marketplace-ui/index.html`)
- Browse 15 expert profiles
- Filter by category (Sales, Engineering, Marketing, etc.)
- Filter by tier (Basic, Enhanced, Premium)
- Search by name or expertise
- Click any card to view full profile

### Profile Pages (`profile-pages/*.html`)

**Basic Tier** (`basic.html`) - FREE
- Simple profile
- What I can help with
- Install button

**Enhanced Tier** (`enhanced.html`) - $49/mo
- Voice samples
- 5 custom tools
- Personality blend
- Testimonials

**Premium Tier** (`premium.html`) - $25,000
- Video introduction
- Complete voice extraction
- 12+ custom tools
- White-glove onboarding
- Private templates
- Calendar booking

### Onboarding Flow (`marketplace-ui/onboarding.html`)
- Step 1: Choose your tier
- Step 2: Configure settings
- Step 3: Test installation
- Step 4: Get started with prompts

### Network Graph (`marketplace-ui/network.html`)
- Visual connection graph
- "People who installed X also installed Y"
- Connection strength indicators
- Expert recommendations
