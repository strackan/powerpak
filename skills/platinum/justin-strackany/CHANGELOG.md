## Changelog Guidelines

### Version Numbers
- **MAJOR** (X.0.0): Breaking changes, major restructuring
- **MINOR** (1.X.0): New features, frameworks, or sections
- **PATCH** (1.0.X): Bug fixes, typos, minor improvements

### Categories
- **Added**: New features, frameworks, or content
- **Changed**: Changes to existing content
- **Deprecated**: Soon-to-be removed features
- **Removed**: Removed features or content
- **Fixed**: Bug fixes, typo corrections
- **Security**: Security-related changes

### Example Entry
```markdown
## [1.1.0] - 2024-12-01

### Added
- New "THE TECHNICAL EXPLAINER" blend recipe
- 3 additional opening templates for technical content
- Case study: Enterprise SaaS launch campaign

### Changed
- Improved SBI feedback framework with more examples
- Updated pricing calculator for 2025 rates

### Fixed
- Typo in M3 template example
- Broken link in customer success section
```
