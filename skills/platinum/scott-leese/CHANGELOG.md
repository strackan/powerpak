## Changelog Guidelines

### Version Numbers
- **MAJOR** (X.0.0): Breaking changes, major restructuring
- **MINOR** (1.X.0): New features, frameworks, or playbooks
- **PATCH** (1.0.X): Bug fixes, typos, minor improvements

### Categories
- **Added**: New features, frameworks, or content
- **Changed**: Changes to existing content
- **Deprecated**: Soon-to-be removed features
- **Removed**: Removed features or content
- **Fixed**: Bug fixes, typo corrections
- **Security**: Security-related changes

### Example Entry
```markdown
## [1.1.0] - 2024-12-05

### Added
- First 90 Days for SDR Managers framework
- Enterprise account planning playbook
- 3 new hiring interview examples (SDR, Sales Manager, VP)

### Changed
- Updated comp plan examples for 2025 market rates
- Enhanced PIP process with more specific metrics

### Fixed
- Typo in MEDDPICC section
- Clarified quota calculation example
```
